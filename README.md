# 🌊 Odyssey

> *"Tell me, Muse, of the man of many ways..."*
> — Homer, *The Odyssey*

When the Trojan War ended, Odysseus did not return home.

Not because he didn't know the way.

Because the journey refused to let him.

For ten long years he crossed the Mediterranean, facing storms sent by Poseidon, outwitting the Cyclops, escaping the Sirens' song, resisting the temptations of Circe and Calypso, descending into the Underworld, and watching ships, hopes, and companions disappear along the way.

Every island taught him something.

Every failure changed him.

Every obstacle demanded that he become wiser than before.

Only then did he reach Ithaca.

---

## ⚓ Our Journey

Programming feels strangely similar.

C is our Mediterranean.

Pointers are our Cyclops.
Manual memory management is Poseidon's wrath.
Segmentation faults are the monsters lurking beneath the waves.
Undefined behavior is the Sirens' song—beautiful until your ship crashes.
Algorithms are the stars we navigate by.
Data structures are the ships carrying us forward.

This repository is my own Odyssey.

Rather than relying on existing libraries, every data structure is implemented from scratch in C—not because it's the easiest path, but because the difficult path teaches the most.

Each module represents another island visited.
Each bug is another storm survived.
Each successful implementation is one step closer to Ithaca.

**Welcome aboard.** ⚓
